SELETOR SIGNIFICADO EXEMPLO
* Seletor universal Aplica-se em todos os elementos no documento *{ margin:0; padding:0; }
Seletor elementos Seleciona elementos como p, h1, span ... p{ margin:0; padding:0; }
Seletor de classe Seleciona um elemento cujo atributo class tem o valor especificado depois do ponto. A classe pode ser utilizado para várias elementos. .classe{ margin:0; padding:0; }
Seletor ID Seleciona um elemento cujo atributo id tem o valor especificado após o símbolo de cerquilha ou jogo da velha .O id é único não deve ser utlizado em mais de uma vez por página. #cabecalho{ margin:0; padding:0; }
Seletor de filho Seleciona um elemento que é filho direto de outro elemento. li > a{ margin:0; padding:0; }
